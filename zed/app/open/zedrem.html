Open Zedrem Folder


Zedrem enables easy editing of files on remote servers. To use it, simply log in to your server (e.g. using SSH) and install the "zedrem" command line tool:

$ curl http://get.zedapp.org | bash

Then, run the following command passing in the directory to edit as the last argument (or omit the directory to edit the current directory):

$ ./zedrem

This will give you a URL you can copy and paste in the input field above.
